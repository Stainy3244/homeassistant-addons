# Changelog

## Initial Release - WORKING HOME ASSISTANT ADD-ON - All further releases will be a straight lift of the upstream

### Added
- Initial release of WatchState Home Assistant addon
- Support for Plex, Jellyfin, and Emby media servers
- Web UI accessible on port 8080
- Configurable UID/GID for file permissions
- Automatic data persistence in Home Assistant config directory
- Watchdog monitoring for addon health
